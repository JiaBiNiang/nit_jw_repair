## NIT教务系统修复插件

解决非IE内核访问教务处出现无法创建对象、权限认证失败等问题。


#### 效果图：

![](img/demo.png)


#### 使用指南：

1、下载release或[百度云:m9z4](https://pan.baidu.com/s/1iiimZT6v-dAKhM6QggaErA)中的nit_jw_repair.zip文件并解压

2、找到Chrome设置->扩展程序。

![](img/step1.png)

3、在页面右上角打开开发者模式

![](img/step2.png)

4、选择加载已解压的扩展程序，选中解压后的nit_jw_repair目录即可

![](img/step3.png)
